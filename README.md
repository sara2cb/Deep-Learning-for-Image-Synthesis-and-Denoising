# Deep-Learning-for-Image-Synthesis-and-Denoising
his project covers the evolution of image manipulation: starting with manual 1D/2D filtering, moving to supervised denoising with encoder-decoders, and concluding with stochastic generation using VAEs. Includes performance benchmarks between ADAM/SGD optimizers and perceptual loss tuning.

## Project Overview
This repository contains a comprehensive pipeline for image analysis and synthesis, implemented in PyTorch. The project is divided into three core modules:

- Classical Vision: Implementation of 2D filtering for edge detection.

- Image Restoration: A 5-level Pyramidal Encoder-Decoder designed for image denoising, comparing neural methods against traditional Gaussian filters.

- Generative Modeling: * Auto-Encoders (AE): Dimensionality reduction and reconstruction on MNIST & CelebA
